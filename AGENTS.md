# AGENTS.md

## Project
AI Blog Automation System

## Goal
検索上位を獲得し、収益化できる高品質ブログ記事を生成する

---

# =========================

# =========================

1. Strategy Agent

3. SEO/AIO Agent
4. Monetize Agent
5. Review Agent

③ テーマ

---

# =========================
# Content Depth Rule（最重要）
# =========================

## 文字数

・1記事：4000〜6000文字  
・H2：300〜600文字  
・H3：200〜400文字  

---

## 深さルール

・結論 → 理由 → 具体例 → 再結論  
・必ず「なぜ？」に答える  
・必ず「どうやる？」に答える  

---

## 必須要素

・具体例（最低2つ）  
・実体験風の説明  
・失敗例  
・改善方法  
・比較  

---

## NG

・1〜2行で終わる見出し  
・抽象的な説明  
・結論だけ  

---
