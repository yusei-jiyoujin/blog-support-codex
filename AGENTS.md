# AGENTS.md

## Project
AI Blog Automation System

## Goal
SEO・AIOに強いブログ記事を安定生成し、WordPressに貼り付けて投稿できる状態にする。

---

# =========================
# Workflow（必須フロー）
# =========================

1. Strategy Agent
2. Writer Agent
3. SEO/AIO Agent
4. Monetize Agent
5. Review Agent

---

# =========================
# Simple Command System
# =========================

ユーザーが以下の形式で入力した場合：

5記事：
① テーマ
② テーマ
③ テーマ
④ テーマ
⑤ テーマ

または

① テーマ
② テーマ
③ テーマ
④ テーマ
⑤ テーマ

これを「5記事作成」として処理する。

---

# =========================
# Article Generation Rule
# =========================

・必ず5記事作成  
・3記事＋2記事で分割  
・各セットごとに完成させる  

---

# =========================
# Strategy Agent
# =========================

役割：
・キーワード設計  
・検索意図分析  
・読者の悩み抽出  
・差別化設計  

出力：
・メインキーワード  
・関連キーワード（5〜10個）  
・検索意図  
・記事構成（H2/H3）  

---

# =========================
# Writer Agent
# =========================

役割：
・記事作成  

条件：
・結論ファースト  
・初心者向け  
・具体例必須  
・Markdown形式  

出力：
・タイトル  
・導入文  
・本文（H2/H3）  
・まとめ  
・Q&A（3〜5個）  

---

# =========================
# WordPress Paste Output Rule
# =========================

## 目的
WordPressブロックエディタにそのまま貼り付け可能な形式で出力する

---

## ルール

・JSON形式は禁止  
・記事は必ず分割して出力  
・コピーしやすい構造にする  
・Markdown形式  
・記事ごとに完全独立  

---
