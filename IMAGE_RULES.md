# IMAGE_RULES.md

## 目的
ブログ記事に合うアイキャッチ画像・記事内画像プロンプトを、SEO・クリック率・読者理解に役立つ形で作成する。

アイキャッチ画像では、記事内容がひと目で伝わるように、記事タイトルまたは画像用短縮タイトルを必ず入れる。  
記事内画像では、読者の理解を助ける図解・フローチャート・比較表・チェックリストを優先する。

---

# =========================
# Eyecatch Image Rule
# =========================

## アイキャッチ画像の目的
記事の内容を一瞬で伝え、クリックされやすくする。

## 基本ルール

・16:9  
・1200×630に近い横長構図  
・記事タイトルまたは画像用短縮タイトルを画像内に必ず入れる  
・文字は1〜2行で読みやすくする  
・スマホでも読める大きさにする  
・文字は太めで視認性を高める  
・背景と文字のコントラストを明確にする  
・シンプル  
・文字を詰め込みすぎない  
・明るく信頼感のある雰囲気  
・ブログ記事のテーマが直感的に伝わる  
・スマホでも見やすい構図  
・情報を詰め込みすぎない  
・誇張しすぎる煽り表現は使わない  
・ビジネスブログ向けの上品で実用的なデザインにする

# =========================
# Eyecatch Text Rule
# =========================

## 目的
ブログ用アイキャッチ画像の実用性を高め、記事内容がひと目で伝わる状態にする。

## ルール

・アイキャッチ画像には、記事タイトルまたは画像用短縮タイトルを必ず入れる  
・基本は画像用短縮タイトルを優先する  
・長い記事タイトルは、意味を変えない範囲で短縮してよい  
・画像用短縮タイトルは20〜35文字程度を目安にする  
・必要に応じて2行構成にしてよい  
・文字は1〜2行で読みやすくする  
・スマホ表示でも読める大きさ・太さにする  
・文字色と背景色のコントラストを明確にする  
・装飾しすぎず、シンプルで信頼感のあるデザインにする  
・読者が一瞬で記事テーマを理解できる表現にする  
・誇張しすぎる煽り文句は使わない  
・ブログ全体の統一感を意識し、同系統の雰囲気で出力する  

## 画像用短縮タイトルの作成ルール

・SEO用の長い記事タイトルとは別に、画像表示用の短縮タイトルを作成する  
・短縮タイトルは20〜35文字程度を目安にする  
・長すぎる場合は2行構成にする  
・「初心者向け」「完全ガイド」「比較」「注意点」「始め方」など、検索意図が伝わる語を優先する  
・冗長な語尾は削ってよい  
・意味が変わる省略は禁止する  
・タイトルだけで記事内容が伝わるようにする  

## 画像用短縮タイトルの例

元タイトル：
もしもアフィリエイトの始め方｜初心者向けに登録から初収益までをやさしく解説

短縮タイトル例：
もしもアフィリエイトの始め方

または

初心者向け  
もしもアフィリエイトの始め方

# Eyecatch Image Prompt Format
# =========================

## 出力形式
アイキャッチ画像プロンプトは、以下の要素を含めて作成する。

・画像比率  
・画像サイズ目安  
・画像内に入れるタイトル  
・背景イメージ  
・人物や作業風景  
・関連アイコン  
・色合い  
・文字の視認性  
・スマホでの読みやすさ  
・クリックされやすさ  
・ブログ向けの雰囲気  

## 基本テンプレート

アイキャッチ画像プロンプト：  
16:9、1200x630相当、記事テーマが一目で伝わるブログ用アイキャッチ画像。  
画像内に「画像用短縮タイトル」を大きく読みやすく入れる。  
明るく信頼感のある配色、シンプルで視認性の高い構図。  
必要に応じてノートPC、作業風景、関連アイコン、チェックリスト、図解要素などを配置する。  
スマホでも文字が読みやすいように、文字は太め・大きめ・高コントラスト。  
クリックしたくなるが誇張しすぎない、上品で実用的なビジネスブログ向けデザインにする。

## アイキャッチ画像プロンプト例

アイキャッチ画像プロンプト：  
16:9、1200x630相当、記事テーマが一目で伝わるブログ用アイキャッチ画像。  
画像内に「もしもアフィリエイトの始め方」というタイトルを大きく読みやすく入れる。  
明るい青と白基調、信頼感のある配色。  
ノートPCでブログ管理画面を開く初心者の作業風景、デスク上にメモ帳とコーヒー。  
スマホでも文字が読めるように、文字は太め・大きめ・高コントラスト。  
シンプルでクリックしたくなるビジネスブログ向けデザイン。

# =========================
# In-Article Image Rule
# =========================

## 記事内画像の目的
読者の理解を助け、記事の滞在時間を伸ばす。

## 基本ルール

・横長16:9  
・図解向き  
・シンプル  
・余白多め  
・日本語ラベルを明確に表示  
・記事内容と一致させる  
・読者が直感的に理解できる構図  
・ブログ記事内に挿入しやすい視認性の高い画像にする  
・ステップ、比較、注意点、流れ、チェックリストを優先する  
・複雑すぎる図解にしない  
・スマホでも見やすい構成にする

# =========================
# In-Article Image Prompt Format
# =========================

## 記事内画像プロンプトに必ず含める要素

記事内画像プロンプトでは、以下をできるだけ具体的に書く。

・画像の比率  
・図解のテーマ  
・表示するステップ  
・各ステップの日本語ラベル  
・アイコンの種類  
・色合い  
・余白  
・読者層  
・ブログ記事内での用途  
・全体の雰囲気  
・情報の流れ  
・横並び、縦並び、フローチャート、比較表などの構図  

## 基本テンプレート

記事内画像プロンプト：  
横長16:9のシンプルな日本語図解。  
「〇〇開始5ステップ」をわかりやすいフローチャートで表現する。  
流れは「ステップ1 → ステップ2 → ステップ3 → ステップ4 → ステップ5」の5段階。  
各ステップに番号付きの丸アイコンを付け、内容に合った小アイコンを配置。  
青と白を基調にしたやさしく見やすい配色、余白多め、初心者向けブログに合うミニマルデザイン。  
日本語ラベルをはっきり表示し、ブログ記事内に挿入しやすい視認性の高い構図。

## 記事内画像プロンプト例

記事内画像プロンプト：  
横長16:9のシンプルな日本語図解。  
「もしもアフィリエイト開始5ステップ」をわかりやすいフローチャートで表現する。  
流れは「ジャンル選定 → ASP登録 → 集客記事作成 → 案件導線設計 → 改善」の5段階。  
各ステップに番号付きの丸アイコンを付け、虫眼鏡、登録フォーム、記事、リンク導線、グラフ改善の小アイコンを配置。  
青と白を基調にしたやさしく見やすい配色、余白多め、初心者向けブログに合うミニマルデザイン。  
日本語ラベルをはっきり表示し、ブログ記事内に挿入しやすい視認性の高い構図。

# =========================
# Article Type Image Rule
# =========================

## 始め方記事
手順・ロードマップ・5ステップ図解を優先する。

### 例
・開始5ステップ  
・初心者ロードマップ  
・登録から実践までの流れ  
・準備物チェックリスト  

---

## 比較記事
比較表、左右比較、無料版と有料版の違いがわかる図解を優先する。

### 例
・無料版 vs 有料版  
・Aサービス vs Bサービス  
・向いている人 / 向いていない人  
・コスト重視 / 時短重視  

---

## 注意点記事
失敗例、チェックリスト、NG例と改善例の図解を優先する。

### 例
・初心者がやりがちな失敗  
・失敗回避チェックリスト  
・NG例 → 改善例  
・注意点7選  

---

## 稼ぎ方記事
収益化までの流れ、案件獲得フロー、収益導線の図解を優先する。

### 例
・収益化ロードマップ  
・案件獲得までの流れ  
・記事作成 → 集客 → 成約  
・収益導線マップ  

---

## おすすめ・レビュー記事
向いている人・向いていない人、メリット・デメリット整理図を優先する。

### 例
・向いている人 / 向いていない人  
・メリット / デメリット  
・おすすめ判断チャート  
・選び方フローチャート  

---

# =========================
# Affiliate Service Image Rule
# =========================

アフィリエイトサービスを扱う記事では、サービスを直接売り込む画像ではなく、読者の理解を助ける画像にする。

## 優先する画像テーマ

・登録手順の図解  
・始め方ロードマップ  
・無料版と有料版の比較図  
・失敗しないチェックリスト  
・収益化までの流れ  
・案件選びの流れ  
・記事作成から成約までの導線  

## ASP記事の画像例

### アイキャッチ
画像内に「〇〇アフィリエイトの始め方」という短縮タイトルを入れる。  
ノートPCでブログ管理画面を開く初心者の作業風景。  
青と白基調、信頼感、シンプルなビジネスブログ向け。

### 記事内画像
「ASP開始5ステップ」をフローチャートで表示。  
ジャンル選定 → 登録 → 集客記事 → 案件導線 → 改善。  
日本語ラベル付き、余白多め、初心者向け。

# =========================
# Text Visibility Rule
# =========================

## 目的
画像内テキストをスマホでも読みやすくする。

## ルール

・文字は太めにする  
・文字サイズは大きめにする  
・背景と文字のコントラストを強くする  
・細い文字や薄い色は避ける  
・長文を入れない  
・1行または2行に収める  
・画像端ギリギリに文字を置かない  
・上下左右に余白を確保する  
・スマホ縮小表示でも意味が伝わるようにする  

---

# =========================
# Brand Consistency Rule
# =========================

## 目的
ブログ全体で統一感のある画像を作る。

## 推奨雰囲気

・明るい  
・やさしい  
・信頼感がある  
・初心者向け  
・清潔感がある  
・ビジネスブログ向け  
・青、白、黒、薄いグレーを基調にする  
・必要に応じてアクセントカラーを少量使う  

## 避ける雰囲気

・派手すぎる  
・広告感が強すぎる  
・怪しい副業感がある  
・文字が多すぎる  
・情報が詰まりすぎている  
・過度に煽るデザイン  

---

# =========================
# NG Rule
# =========================

・文字を詰め込みすぎない  
・過度に広告っぽくしない  
・誇大表現を入れない  
・記事内容と関係ない画像にしない  
・複雑すぎる図解にしない  
・読者が一目で理解できない構図にしない  
・タイトルと画像内容がズレないようにする  
・読みにくい小さい文字にしない  
・背景と文字のコントラストを弱くしない  
・スマホで読めない画像にしない  
・サービスロゴや商標を不自然に強調しすぎない  
・収益を過度に煽る表現を使わない  

---

# =========================
# Final Output Rule
# =========================

記事作成時には、以下を必ず出力する。

## 画像用短縮タイトル
記事タイトルが長い場合、アイキャッチ用に短くしたタイトルを出力する。

## アイキャッチ画像プロンプト
画像内に画像用短縮タイトルを入れる前提で作成する。

## 記事内画像プロンプト
記事内容を理解しやすくする図解プロンプトを作成する。
