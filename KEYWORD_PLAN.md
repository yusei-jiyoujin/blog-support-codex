# KEYWORD_PLAN.md

## 目的
SEOに強く、重複しないキーワード設計を行う。

---

# =========================
# 基本ルール
# =========================

・1記事1メインキーワード  
・検索意図を重複させない  
・同じサービス名でも切り口を変える  
・収益記事と情報記事を混ぜる  

---

# =========================
# キーワード分類
# =========================

## 始め方系
例：
・Canva Pro 始め方  
・ConoHa WING ブログ 始め方  
・A8.net 登録 方法  

## 稼ぎ方・活用系
例：
・Canva Pro 副業 稼ぎ方  
・ConoHa WING ブログ 収益化  
・A8.net アフィリエイト 稼ぎ方  

## 比較系
例：
・Canva 無料版 Pro 違い  
・ConoHa WING エックスサーバー 比較  
・A8.net もしもアフィリエイト 比較  

## 注意点・失敗系
例：
・Canva Pro 注意点  
・ConoHa WING デメリット  
・A8.net 審査 落ちる  

## おすすめ・評判系
例：
・Canva Pro おすすめ  
・ConoHa WING 評判  
・A8.net 初心者 おすすめ  

---

# =========================
# 1サービス展開ルール
# =========================

1つのアフィリエイトサービスだけでも、以下の5方向へ展開する。

1. サービス名 + 始め方
2. サービス名 + 稼ぎ方 / 活用方法
3. サービス名 + 比較 / 違い
4. サービス名 + 注意点 / デメリット
5. サービス名 + おすすめ / 評判

---

# =========================
# NGルール
# =========================

・同じキーワードを複数記事で使わない  
・検索意図が近すぎるテーマを連続させない  
・すべてを収益キーワードにしない  
・サービス名だけの薄い記事にしない  

---

# =========================
# 目的
# =========================

・検索意図の重複を防ぐ  
・内部リンクしやすくする  
・SEO評価を分散させない  
・1サービスから5記事を自然に作れるようにする
